<h3>Tech Stack</h3>
<p>Status is fullstack, fully-featured social media platform build using React, GraphQL, Node and MongoDB. 

</p>


<h3>Features</h3>

 - User Register/Login
 - Create a post
 - Like/Unlike Posts
 - Comment on Posts
 - Like/Unlike Comments
 - Local Storage for Authentication Token
 - Display like and comment count on post
 - React Routing
 - Protected Routes
 - Error Handing
 - Form Validations
